## 本地预览

```bash
npx serve .
```

## 添加新项目

1. 创建项目文件夹：`mkdir project-name`
2. 添加 `index.html` 到文件夹中
3. 主页会自动检测并展示新项目

## 技术栈

- 纯静态 HTML
- Tailwind CSS (CDN)
- 原生 JavaScript
