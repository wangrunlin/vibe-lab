01 / PURPOSE & VALUE

核心价值

01

极简接入

通过简单的 URL 参数即可发起支付，无需每个项目单独集成 Stripe SDK，零代码负担。

02

快速验证

适用于预售验证付费意愿、个人赞赏及产品一次性购买场景。先收款，再开发。

03

全环境通用

无论是静态博客、Notion 页面还是社交媒体，任何能放链接的地方都能变成收银台。

Deployment: Cloudflare Workers
02 / INTERFACE

支付跳转逻辑

ENDPOINT
GET /pay

Required Parameters

amount
Number
金额 (9.9 - 999)
product
String
产品/服务名称

Optional Parameters

currency Default: "usd"
email Pre-fill user email
success_url Return after success
metadata JSON extra info

系统自动处理 Stripe Checkout Session 创建并进行 302 重定向。错误时将显示引导页面。
